## 算法1
##### 线性dp
f[i]表示i时间内完成的最大值，注意外层虽然n循环，内层循环中，每个任务只遍历了一次，因此时间复杂度为 O(n + m);

#### 时间复杂度
O(n + m)

#### 参考文献

#### C++ 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
#define x first
#define y second
typedef pair<int, int> PII;

const int N = 1e6 + 10, M = 1010;

int n, m, r;
vector<PII> p[N];
int f[N];

int main() {
    cin >> n >> m >> r;
    for (int i = 1; i <= m; i++) {
        int a, b, c;
        cin >> a >> b >> c;
        p[b].push_back({a, c});
    }
    for (int i = 0; i <= n; i++) {
        f[i] = f[i - 1];
        for (auto [l, c]: p[i]) {
            f[i] = max(f[i], f[max(l - r, 0)] + c);
        }
    }
    cout << f[n] << endl;
}
```
