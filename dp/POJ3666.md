## 算法1
##### 线性dp + 离散化
状态表示：f[i][j]
集合：所有将前i个数变为符合题目要求(增或减)的序列，且第i个元素变为b[j]的总代价
属性：Min

状态计算：
    for (i -> [1, n] ) 从 1, n 循环 i
        for (j -> [1, m] )
            for (k -> [1, j] )
                f[i][j] = min(f[i][j], f[i - 1][k] + abs(a[i] - b[j]) );
    简化一下为：
        f[i][j] = min(f[i][j], min(f[i - 1][k]) + abs(a[i] - b[j]));
    如此时间复杂度为 O(n^3)
因此类似最长公共上升子序列简化为O(n^2)
    可以发现：f[i - 1][k]每次比较 j 前面的都是重复比较，，我们可以
    令一个 num 维护 min(f[i - 1][k]) 1 <= k <= j ，每次新加入值 将num更新一下就可以
        f[i][j] = min(f[i][j], num + abs(a[i] - b[j]);

#### 时间复杂度
O(n2)

#### 参考文献

#### C++ 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long

const int N = 2010, INF = 1e15;

int n, m;
int f[N][N], a[N], b[N];

int calc(){
    memset(f, 0x3f, sizeof f);
    for (int i = 1; i <= m; i++) f[0][i] = 0;
    for (int i = 1; i <= n; i++) {
        int num = INF;
        for (int j = 1; j <= m; j++) {
            num = min(num, f[i - 1][j]);
            f[i][j] = min(f[i][j], num + abs(a[i] - b[j]));
        }
    }
    int res = INF;
    for (int i = 1; i <= m; i++) res = min(res, f[n][i]);
    return res;
}

signed main() {
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        b[i] = a[i];
    }
    sort(b + 1, b + n + 1);
    m = unique(b + 1, b + n + 1) - (b + 1);
    int res1 = calc();
    reverse(b + 1, b + m + 1);
    int res2 = calc();
    cout << min(res1, res2) << endl;
    return 0;
}

```
