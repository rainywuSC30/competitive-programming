## 算法1
##### 排序

1. 如果 $max(a_i, a_j) == max(b_i, b_j)$  不考虑a，b内的逆序对，显然 $a_i + a_j$ 和 $b_i + b_j$中小的应该排在前面

2. 如果 $max(a_i,a_j) > max(b_i, b_j)$ 
   直觉上 a应该排在b的后面

   * 如果b排在后面 $b_i,b_j$ 和 a数组的大值形成 2个逆序对
   * 如果 a数组中的小值比 $b_i,b_j$都小，则b在后面相比在前面会少2个逆序对
   * 如果 a数字中的小值在 $b_i, b_j$之间，则b在后面相比在前面逆序对不增

   综上，b排在后面不会使逆序对减少，反而可能增加，因此要 b要排在a的前面

3. 如果 $max(a_i,a_j) < max(b_i, b_j)$  a排在b的前面

#### C++ 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
#define endl '\n'
#define x first
#define y second
typedef long long LL;
typedef pair<int, int> PII;

const int N = 2e5 + 10, M = 4e5 + 10, INF = 0x3f3f3f3f;

int n;
PII a[N];

bool cmp(PII a, PII b){
    int ax = a.x, ay = a.y, bx = b.x, by = b.y;
    if (max(ax, ay) == max(bx, by)) {
        return ax + ay < bx + by;
    }
    else if (max(ax, ay) > max(bx, by)) {
        return 0;
    }
    else return 1;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while(T--) {
        cin >> n;
        for (int i = 1; i <= n; i++) {
            cin >> a[i].x >> a[i].y;
        } 
        sort(a + 1, a + n + 1, cmp);
        for (int i = 1; i <= n; i++) {
            cout << a[i].x << " " << a[i].y << " ";
        }
        cout << endl;
    }
    return 0;
}
```
